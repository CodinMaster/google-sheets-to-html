# google-sheets-to-html
Import data from Google Sheets and render as custom HTML Page.

---
Make the spreadsheet publically visible in Google Sheets. Just paste its url in '$spreadsheet' variable.

Demo Spreadsheet: https://docs.google.com/spreadsheets/d/1RbHac8SCA1Tx_F0dKtjk7NZbF8yBuOgcX0q-JFhtpa4

**Input:** 
![google sheets](https://i.imgur.com/QFWT9KN.png)

**Output:**
![html page](https://i.imgur.com/TQd1B1c.png)

Presently tables styled by Bootstrap 4, custom styling can be applied.

Advanced:
As per project requirements, instead of tables other elements like cards,etc. can also be used thereby using Google Sheets as a Database
(ideal for easy updation of data by non-technical people).
